# Homework 4 Client TSX Template

This is the TypeScript/TSX version of the Homework 4 client template.

It includes the stock list fetch example used by Homework 4 so the dropdown can load all available tickers from the local API server.

## Run It

```bash
npm install
npm run dev
```

Make sure the Homework 4 server is running on `http://localhost:8000` so the dropdown can fetch the stock list.
