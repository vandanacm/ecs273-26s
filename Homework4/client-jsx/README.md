# Homework 4 Client JSX Template

This is the JavaScript/JSX version of the Homework 4 client template.

It includes the same stock list fetch example as the TypeScript client, but with `.jsx` source files and without the TypeScript config files.

## Run It

```bash
npm install
npm run dev
```

Make sure the Homework 4 server is running on `http://localhost:8000` so the dropdown can fetch the stock list.
